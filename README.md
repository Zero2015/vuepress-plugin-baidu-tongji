# vuepress 百度统计

> Baitu tongji plugin for vuepress

## Install

```bash
yarn add -D @zenghp2015/vuepress-plugin-baidu-tongji
# OR npm install -D @zenghp2015/vuepress-plugin-baidu-tongji
```

## Usage

```js
module.exports = {
  plugins: [
    '@zenghp2015/vuepress-plugin-baidu-tongji', 
    { hm: 'xxx'}
  ]
}
```

## Options

### hm

- Type: `string`
- Default: `undefined`
